# Lundun Audio Website

Client website developed for Lundun Audio, a brand focused on music production, sound engineering, and audio-related services.

This project involved designing and implementing a responsive website to showcase the client's brand, services, and online presence.

---

## Live Website

https://lundunaudio.com

---

## Project Overview

The Lundun Audio website was created to provide a professional online presence for the client, allowing visitors to learn about the brand, explore services, and easily get in touch.

The focus of the project was on:

- Clean visual design
- Mobile responsiveness
- Clear presentation of services
- Fast loading performance
- Simple navigation for users

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive design principles
- Basic SEO optimization

---

## Features

- Fully responsive layout for desktop and mobile devices
- Structured content layout for easy navigation
- Contact section for client inquiries
- Clean and minimal UI focused on audio branding
- Performance-focused front-end design
